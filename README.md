# wjivan.github.io

Personal portfolio site (GitHub Pages) for **Wen Jian**.

## Local preview
This is a static site. For a quick local preview:

```bash
cd wjivan.github.io
python3 -m http.server 8000
# then open http://localhost:8000
```

## Structure (v1)
- `/` – Home
- `/projects/` – Projects
- `/writing/` – Writing (placeholder)
- `/about/` – About (placeholder)

## Custom domain (optional)
When ready to use `wenjian.dev`, we’ll:
- add a `CNAME` file with `wenjian.dev`
- update DNS at the registrar (IONOS)
- enable HTTPS in GitHub Pages

(We haven’t applied DNS changes yet.)
